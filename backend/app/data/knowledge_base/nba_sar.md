# NBA Self-Assessment Report (SAR) Framework

## Criterion 1: Vision, Mission and PEOs
- **PEO1**: Professional Excellence (Career success).
- **PEO2**: Innovation & Entrepreneurship.
- **PEO3**: Continuous Learning.
- **Mapping**: PEOs mapped to POs (1-12).

## Criterion 2: Program Curriculum and Teaching-Learning
- **Curriculum**: Basic Sciences (17%), Engineering (20%), Core (40%), Electives (16%), Projects (7%).
- **Revision**: Every 2 years with Industry/Alumni input.
- **Pedagogy**: Flipped Classroom (20%), PBL (15%), Lectures (40%).
- **Assessment**: CIA (30%) + ESE (70%).
- **CO Attainment**: Direct (80%) + Indirect (20%). Targets set (Level 3 > 70%).

## Criterion 3: Course Outcomes (COs)
- Defined for every course.
- Mapped to POs/PSOs.
- Evaluated via Tests, Assignments, Exams.

## Criterion 4: Students' Performance
- Enrollment Ratio.
- Success Index (Pass percentage).
- Academic Performance (CGPA).
- Placement, Higher Studies, Entrepreneurship.

## Criterion 5: Faculty Information
- Student-Faculty Ratio (SFR) target 1:15 or 1:20.
- Faculty Cadre Ratio.
- Faculty Qualification (Ph.D.).
- Retention.

## Criterion 6: Facilities and Technical Support
- Labs equipped with software (Matlab, Visual Studio).
- Technical staff availability.
- Maintenance.

## Criterion 7: Continuous Improvement
- PO/PSO attainment calculation.
- Actions taken for improvement (closing the loop).
- Academic Audit.

## Criterion 8: First Year Academics
- First Year Student-Faculty Ratio.
- First Year Academic Performance.

## Criterion 9: Student Support Systems
- Mentoring.
- Feedback analysis.
- Co-curricular activities.

## Criterion 10: Governance, Institutional Support and Financial Resources
- Budget allocation.
- Library.
- Internet.
- Career Counseling.
